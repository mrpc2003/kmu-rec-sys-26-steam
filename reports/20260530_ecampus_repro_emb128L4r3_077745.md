# eCampus 재현성 패키지 — emb128_L4_reg1e-3 4-seed LightGCN 앙상블 public 0.77745

대회 규칙: 최종 선택 제출 2건은 **동일 결과가 나오는 코드 및 조건**을 eCampus로 제출해야 하며,
복원되지 않으면 제출로 인정되지 않는다. 본 문서는 **최종 최고 후보(public 0.77745)**의 재현 절차를 기술한다.

## 0. 재현성 증거 (이미 검증됨)

집계 단계는 결정적: 동일한 4개 seed의 raw test score로 재집계하면 제출 파일과 **byte 단위로 동일**(SHA256 일치, 재실행 확인).

검증 negative 샘플러 과적합이 구조적으로 불가능(stacker 0.75355 회귀와 대조).

## 3. 재현 명령 (2단계)

### 3-a. 4개 seed full-train → test raw score (병렬 권장, seed당 ~1760s)
```bash
cd /opt/data/kaggle/kmu-rec-sys-26-steam
for SEED in 42 123 2024 7; do
  env -u VIRTUAL_ENV UV_NO_ACTIVE_VENV=1 \
    uv run --extra-index-url https://download.pytorch.org/whl/cu128 \
    --with 'torch==2.10.0+cu128' --with numpy --with pandas --with scipy --with scikit-learn \
    python scripts/lightgcn_fulltest_param.py \
      --emb-dim 128 --n-layers 4 --reg 1e-3 --seed ${SEED} --device cuda:0 \
      --out-dir artifacts/lightgcn_emb128L4r3_fulltest/seed${SEED}
done
# 출력 per seed: artifacts/lightgcn_emb128L4r3_fulltest/seed{SEED}/test.csv (col: score_lightgcn)
```

### 3-b. 4-seed raw score 평균 → per-user top-half → 제출 후보 (SHA 검증 포함)
```bash
env -u VIRTUAL_ENV UV_NO_ACTIVE_VENV=1 \
  uv run --with numpy --with pandas --with scipy --with scikit-learn \
  python scripts/emb128L4r3_test_aggregate.py
# 출력: artifacts/lightgcn_emb128L4r3_fulltest/test_candidate/candidate_lightgcn_emb128L4r3_seed_ens.csv
#       (SHA 7e3191de... 재현 확인됨)
```

## 4. 모델 / 하이퍼파라미터 (4 seed 공통)

| 항목 | 값 |
|---|---|
| 모델 | LightGCN (feature transform/nonlinearity 없음) |
| 손실 | BPR (logsigmoid pairwise) |
| emb_dim | **128** |
| n_layers | **4** |
| lr | 1e-3 (Adam, weight_decay=0) |
| reg | 1e-3 (L2 on embeddings, batch 정규화) |
| batch_size | 4096 |
| epochs | 200 |
| seeds | **42, 123, 2024, 7** (numpy default_rng + torch.manual_seed) |
| 학습 | canonical `train_lightgcn` (lightgcn_train.py); seed당 ~1760s, full-train final_loss 0.319~0.321 |

## 5. 앙상블 / 디코딩

- 각 seed별 LightGCN을 위 config로 full train, test pair에 user/item 임베딩 내적으로 `score_lightgcn` 부여
- 4개 seed의 **raw score 평균**: `score_ens = mean(seed42, seed123, seed2024, seed7)`
- **유저별 top-half**: 후보 수 n인 유저는 `score_ens` 상위 n/2개를 `1`, 나머지 `0` (테스트셋 유저별 1:1)
- 구현: `recsys_played_utils.predict_tophalf`

- Public LB: **0.77745** (최종 best)

- Validation (uniform = public surrogate): emb128 4-seed 앙상블 **0.76505** (emb64 앙상블 0.76145 +0.00360)
- 전이 확증: uniform Δ(vs emb64) +0.0036 → public Δ +0.0062 (전이비 1.72); anchor 기준 uniform Δ +0.0106 → public Δ +0.0150 (전이비 1.42). surrogate가 emb64·emb128 **두 번 연속** 예측 적중.
